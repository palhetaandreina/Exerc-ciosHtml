Listas com HTML

Para listas ordenadas usamos a tag <ol>listas ordenadas</ol> e <ul>listas não ordenadas</ul> Exemplo:

  1. Item 1
  2. Item 2
    1. item 2.1
    2. item 2.2
    3. item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Exemplo de listas não ordenadas

  • item 1
    • item 2
    • item 3

Lista de definições

Toda lista de definições está dentro de uma tag <dl></dl> (definition list) cada termo é um <dt></dt> (Definition term) e cada descrição é um <dd></dd> (definition description). Assim como os itens da lista essas duas tags possuem fechamento opcional, segundo a referencia oficial da HTML5
Exemplo:

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
JavaScript
Linguagem de programação para criar interatividade em sites.
